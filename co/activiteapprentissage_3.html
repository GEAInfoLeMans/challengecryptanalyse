Implémentation du chiffre de César

Pour cette implémentation de notre premier algorithme de chiffrement, nous avons besoin de plusieurs fonctions Excel destinées à la manipulation du texte.

Il s'agit en effet de découper le texte à chiffrer, puis d'appliquer le décalage à chaque lettre, et enfin fusionner le texte.

Pas de panique cependant, Excel va nous permettre de travailler cela sans trop de difficulté.

ÉTAPE 1 : DÉCOUPAGE DU TEXTE

Pour découper notre texte, nous avons besoin de trois fonctions :

  • GAUCHE : https://support.office.com/fr-fr/article/GAUCHE-GAUCHEB-fonctions-9203d2d2-7960-479b-84c6-1ea52b99640c

  • RIGHT : https://support.office.com/fr-fr/article/DROITE-DROITEB-fonctions-240267ee-9afa-4639-a02b-f19e1786cf2f

  • NBCAR : https://support.office.com/fr-fr/article/NBCAR-LENB-fonctions-29236f94-cedc-429d-affd-b5e33d2c67cb

Exemples :

Fonction

Résultat

Gauche("Ceci est un message" ; 1)

"C", la première lettre du message

Gauche("Ceci est un message" ; 4)

"Ceci", les quatre premières lettres du message

Droite("Ceci est un message" ; 7)

"message", les 7 dernières lettre du message

Droite("Ceci est un message" ; 18)

"eci est un message", les 18 derniers caractères

NBCHAR("Ceci est un message")

19, le nombre de caractères du message

En utilisant les fonctions dessus et en vous aidant des exemples, reproduisez le tableau Excel présenté à gauche.

Pour cela, entrez en B2 le message à découper.

Les valeurs des cellules de B3 à B20 sont ensuite obtenues avec la même formule qui est tirée.

Les valeurs des cellules de A2 à A20 sont également obtenues avec une seule formule qui est tirée.

ÉTAPE 2 : CHIFFREMENT

Maintenant que nous sommes capables de découper notre texte, nous allons voir comment appliquer un décalage à une lettre. Pour cela, il vous faut savoir que chaque lettre est en fait représentée par Excel via un chiffre, une sorte de code qui l'identifie. C'est sur ce code que nous allons appliquer nos opérations.

Pour cela, nous avons besoin de trois fonctions :

  • Code : https://support.office.com/fr-fr/article/CODE-CODE-fonction-c32b692b-2ed0-4a04-bdd9-75640144b928

  • Majuscule : https://support.office.com/fr-fr/article/MAJUSCULE-MAJUSCULE-fonction-c11f29b3-d1a3-4537-8df6-04d0049963d6

  • Car : https://support.office.com/fr-fr/article/CAR-CAR-fonction-bbd249c8-b36e-4a91-8017-1c133f9b837a

Exemples :

Fonction

Résultat

Code("A")

65

Code("A") - 64

1

CODE(MAJUSCULE("a")) - 64

1

CAR(65)

"A"

CAR(66)

"B"

A l'aide d'une ou plusieurs fonctions ci-dessus, réaliser le tableau de droite. La colonne de gauche est entrée dans Excel, la colonne de droite est obtenue en utilisant une formule.

Vous pouvez maintenant réaliser le tableau présenté à droite.

Pour cela, entrer le texte en clair dans la colonne A, puis en utilisant des formules, déduisez en la colonne B. En appliquant le décalage que vous aurez entré en colonne C, obtenez automatiquement la colonne D.

ÉTAPE 3 : CONCATÉNATION DU MESSAGE CHIFFRÉ

La dernière étape consiste à concaténer le texte de la colonne D, à savoir mettre bout à bout toutes les lettres. Pour cela, Excel nous propose la fonction CONCATENER : https://support.office.com/fr-fr/article/CONCATENER-CONCATENER-fonction-8f8ae884-2ca8-4f7a-b093-75d702bea31d
